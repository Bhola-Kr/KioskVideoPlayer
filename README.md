# Video & Image Kiosk App

A simple Android **kiosk-style application** that plays **videos and images in a continuous loop** from local device storage.  
Ideal for **digital signage**, **advertising screens**, and **information kiosks**.

---

## Features

- Plays videos and images automatically
- Continuous loop playback
- Works with local storage media
- Simple and user-friendly interface
- Alerts when no media files are found
- Designed for unattended kiosk usage

---

## How It Works

1. Install the app on an Android device  
2. Copy video or image files to the device storage  
3. Open the app and tap **Play Kiosk**  
4. Media starts playing automatically in a loop  

No internet connection is required.

---

## Supported Media

- **Images:** JPG, PNG, JPEG  
- **Videos:** MP4, MKV, 3GP  

(Supported formats may vary by device)

---

## Use Cases

- Retail advertising displays  
- Exhibition and event screens  
- Restaurant menu boards  
- Office or school notice boards  
- Public information kiosks  

---

## Requirements

- Android device (Android 6.0 or higher recommended)
- Storage access permission
